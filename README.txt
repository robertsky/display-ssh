== Description ==

A simple plugin to show public keys of the authors.
You can send a plain-text of your ssh key(s) to your friends/colleagues using a predefined link.

The SSH Key field to enter is in Your Profile.

An example of the predefined link: http://example.com/author/<username>/ssh/

== Installation ==

1. Upload `display-ssh.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Key in your ssh key in 'Your Profile' Menu
